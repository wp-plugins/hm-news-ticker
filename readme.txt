== Description ==

Hm News Ticker is a nice & awesome news ticker with controls. You can easily install and customize it by yourself. Here is nice controls for your news post or blog post headlines. The plugin is very much light weight. It has linking system so that reader can easily go there desire post.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_shortcode('[YOUR SHORTCODE]'); ?>` in your templates

Hm News Ticker is a nice & awesome news ticker with controls. You can easily instal and customize it by yourself. 
See below for the custom options:



<strong>General Ticker</strong>
<pre>[ticker_list id="1"]</pre>

<strong>Category</strong>
<pre>[ticker_list id="2" category="category_slug"]</pre>

<strong>Speed</strong>

<pre> [ticker_list id="3" speed=".03"] </pre>

<strong>Title</strong>

<pre>[ticker_list id="4" title="Breaking"]</pre>

<strong>Color</strong>

<pre>[ticker_list id="5" color="#F075C9"]</pre>

<strong>Background</strong>

<pre>[ticker_list id="6" bg="#2B3039"]</pre>

<strong>Quantity</strong>
<pre>[ticker_list id="7" quantity="1"]</pre>

<strong>For controls</strong>

<pre>[ticker_list id="8" control="false"]</pre>

Note: If you want to change your different parameters , you must change your <pre>id</pre> .
